Nontice
=======
### non-server notice for GameMaker Studio 2


## How to use it?

### Web Post    
||||| Notice Context |||||   

    Put what you want between ||||| and |||||

### Create
nontice_get(self, {blog_id}, {log_no});    

    {blog_id} naver blog id   
    {log_no} blog post id   

### HTTP
nontice_success(self);   

    return notice text on response
